## Marvel Avengers

<p>Marvel Movies List for Avengers</p>

## Requirements

- Node.JS 6 or 8x
- GIT to clone this Respo

## Installation

- Install via GIT

```
git clone https://github.com/mskian/marvel-avengers.git
cd marvel-avengers
npm install
```

- Execute the Script

```
node app.js
```
