## Requirements

- Node.JS 8x
- GIT to clone this Respo

## Installation

- Install via GIT

```bash
git clone https://github.com/mskian/marvel-avengers.git
cd marvel-avengers
npm install
```

- Execute the Script

```bash
node app.js
```

## Install Via NPM

- Install and Execute

```bash
npm install -g marvel-avengers
```

```bash
marvel
```

## Python CLI

```bash
python marvel.py
```
